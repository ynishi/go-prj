# Go Project template for executable. 

* Go version is 1.8
* License is Apache-2.0 

## Install

```bash
$ go get -d "github.com/ynishi/go-prj"
$ cd $GOPATH/src
$ mv github.com/ynishi/go-prj /path/to/your/prj
$ cd /path/to/your/prj
$ make setup
# remove setup for Makefile
$ vi Makefile
$ git init
```
